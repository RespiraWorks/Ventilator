## Assembly

**Note: Please see general advice in the [manufacturing methods guide](../../methods).**

The oxygen regulator requires 2x of 1/4"ID tubing adapters (**A2**) to interface with tubing.

Mounts to acrylic with 2x:
* M4x14 screws
* M4 washers 9mm OD
* M4 lock nuts
